# Odin Recipes

This project is part of The Odin Project Foundations course.

## Description
A basic HTML website that showcases a few recipes.
The goal of this project is to practice HTML structure, links, lists,
and Git/GitHub workflow.

## Skills Demonstrated
- HTML boilerplate
- Headings and paragraphs
- Lists (ul, ol)
- Links and images
- Project structure
- Git and GitHub
